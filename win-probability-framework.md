# 获奖概率评估框架

> 2026-02-21 | 这是一个严肃的分析工具，不是拍脑袋的数字

## 为什么这件事很重要

中国创作者花了钱、花了时间、克服了语言壁垒去投稿——如果一点收获都没有，他们不会再来。我们的核心承诺不是"保证获奖"，而是"最大化你的投入产出比"。这意味着我们需要：

1. 诚实地评估每个竞赛的获奖难度
2. 为不同水平的创作者推荐不同层级的竞赛
3. 帮助创作者建立合理预期

**关键发现：**

- 专业组（Sony Pro）投稿量远小于公开组（1,000 vs 430,000）
- 小型竞赛（150-350篇）获奖率可达 1-2%

## 获奖概率的多维评估模型

### 不能只看一个维度

你说得对——获奖概率高的判定依据不能太单一。"投稿少所以容易赢"是最常见的误判。一个只有 50 人投稿的竞赛，如果 50 人都是 MFA 毕业生，比 5,000 人投稿但大部分是业余爱好者的竞赛更难赢。

### 六维评估模型

```
获奖概率综合评分 = f(竞争密度, 竞争者质量, 评审偏好匹配, 作品适配度, 结构性优势, 历史数据)
```

#### 维度一：竞争密度（Competition Density）
- 投稿总量（如果公开的话）
- 投稿费作为过滤器的效果（免费 vs $50）
- 是否限制投稿数量（限额300篇 vs 无限制）
- 地理/资格限制（仅限英国 vs 全球开放）

**评分逻辑：**
| 投稿量估算 | 密度评分 |
|-----------|---------|
| <100 | 9/10 |
| 100-500 | 7/10 |
| 500-2,000 | 5/10 |
| 2,000-10,000 | 3/10 |
| >10,000 | 1/10 |

#### 维度二：竞争者质量（Competitor Quality）
这是最难量化但最重要的维度。

**信号指标：**
- 投稿费高低（高费用 → 更认真的投稿者 → 质量更高）
- 是否限制"新人"（新人赛 → 质量相对低）
- 往届获奖者的背景（MFA毕业生 vs 业余爱好者）
- 竞赛声望（声望越高 → 吸引的高手越多）
- 是否有"专业组"和"业余组"分开评审

**评分逻辑：**
| 竞争者质量信号 | 质量评分 |
|--------------|---------|
| 新人专属 + 免费 + 低声望 | 2/10（容易）|
| 开放 + 低费用 + 中等声望 | 4/10 |
| 开放 + 中等费用 + 高声望 | 6/10 |
| 开放 + 高费用 + 顶级声望 | 8/10（很难）|
| 仅限已出版作者 + 顶级声望 | 10/10（极难）|

#### 维度三：评审偏好匹配（Judge-Work Fit）
- 评委的已知偏好（从评语、往届选择中推断）
- 竞赛的风格倾向（文学性 vs 商业性 vs 实验性）
- 主题匹配度
- 是否有"国际多样性"偏好（很多竞赛明确寻求非英语母语国家的声音）

**这是我们的核心价值所在——帮创作者找到"评委会喜欢你的作品"的竞赛。**

#### 维度四：作品适配度（Work-Competition Fit）
- 字数是否在最佳范围内（太短或太长都不利）
- 类型/风格是否匹配
- 主题是否契合
- 语言质量（翻译作品 vs 原创英语作品）

#### 维度五：结构性优势（Structural Advantages）
这是中国创作者特有的分析维度：

| 因素 | 对中国创作者的影响 |
|------|------------------|
| 无语言要求（摄影/设计/插画） | 🟢 巨大优势——消除最大壁垒 |
| 支持中文投稿（KADOKAWA） | 🟢 巨大优势 |
| 明确寻求"国际声音" | 🟢 优势——中国视角是稀缺资源 |
| 有"新人"类别 | 🟢 优势——降低竞争强度 |
| 有"学生"类别 | 🟢 优势——降低竞争强度 |
| 需要英语母语水平的文学性 | 🔴 劣势——翻译作品很难达到 |
| 需要PayPal/国际信用卡 | 🟡 障碍——但可解决 |
| 需要英文Bio/Cover Letter | 🟡 障碍——我们可以帮写 |

#### 维度六：历史数据（Historical Data）
- 往届是否有中国/亚洲获奖者
- 往届获奖者的背景多样性
- 竞赛是否在扩大国际参与度

## 综合评分公式

```
获奖概率评分 = (
    竞争密度 × 0.20 +
    竞争者质量(反向) × 0.15 +
    评审偏好匹配 × 0.25 +
    作品适配度 × 0.20 +
    结构性优势 × 0.10 +
    历史数据 × 0.10
) / 10

输出：0-10 分
- 8-10: 高概率（推荐优先投）
- 5-7: 中等概率（值得尝试）
- 3-4: 低概率（挑战性投稿）
- 1-2: 极低概率（仅限顶尖作品）
```

**注意：评审偏好匹配权重最高（0.25），因为这是最能被我们优化的维度。**

## 诚实的风险提示

我们必须对用户诚实：

1. **没有任何竞赛是"保证获奖"的。** 即使是最小的竞赛，获奖也需要真正好的作品。
2. **翻译作品在纯文学竞赛中天然劣势。** 除非翻译质量极高，否则很难与英语母语作者竞争"语言的音乐性"。
3. **摄影/设计/插画是中国创作者的最佳切入点。** 无语言壁垒，纯粹比拼作品质量。
4. **闪小说（300字以内）是文学类的最佳切入点。** 字数少 → 翻译难度低 → 质量更可控。
5. **投稿是一个概率游戏。** 投 10 个竞赛比投 1 个竞赛的期望收益高得多。我们的价值是帮用户选对那 10 个。

## 数据收集计划

要让这个框架真正有用，我们需要持续收集：

- [ ] 每个竞赛的投稿量（很多不公开，需要从社交媒体、往届报道中推断）
- [ ] 往届获奖者的国籍分布
- [ ] 评委的评语和偏好分析
- [ ] 用户投稿后的实际结果（我们自己的数据，最有价值）
- [ ] 竞赛方的反馈（远期：与竞赛方建立关系）

**最终目标：每个竞赛都有一个基于真实数据的"中国创作者获奖概率评分"。** 这是我们最核心的数据资产，也是竞品无法复制的壁垒。
